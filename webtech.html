Web Technology - 001

Syllabus

  1. Introduction to Web Technology
  2. HTML5 & CSS3
  3. JavaScript Basics and DOM Manipulation
  4. Client-Server Architecture
  5. HTTP Protocol and Web Servers
  6. Frontend Frameworks (Basics of React/Angular)
  7. Server-side Scripting with Node.js / PHP
  8. Databases: MySQL & MongoDB
  9. Web Hosting & Deployment
  10. Security in Web Applications

Explanation of Syllabus

This course introduces the fundamental concepts and tools used in developing modern web applications. It starts with the structure and design of web pages using HTML and CSS, moves through interactive front-end programming with JavaScript, and explores the backend with Node.js or PHP. Students will also gain knowledge about client-server architecture, protocols like HTTP, basic database operations, and deployment strategies for real-world applications.

Lab Manual with Explanation

  1. Lab 1: Basic HTML page
    Create a simple webpage with headings, paragraphs, and links.
  2. Lab 2: CSS Styling
    Apply styles using internal and external CSS files.
  3. Lab 3: Forms and Tables
    Create registration form and display data in a table.
  4. Lab 4: JavaScript Basics
    Use variables, functions, and events in JavaScript.
  5. Lab 5: DOM Manipulation
    Dynamically change content and styles using JavaScript DOM.
  6. Lab 6: Node.js Server
    Create a simple HTTP server using Node.js.
  7. Lab 7: Database Connection
    Connect a web page to MySQL or MongoDB.
  8. Lab 8: Hosting
    Deploy a site using GitHub Pages or Netlify.

Class Notes

  • HTML uses tags to build content structure.
  • CSS styles your HTML using selectors and properties.
  • JavaScript makes your page interactive (clicks, form validation, etc).
  • Frontend is what users see; backend handles logic & database.
  • Web servers deliver files; HTTP is the language browsers use to talk to them.
  • Frameworks like React simplify UI development with components.
  • Node.js lets you use JavaScript on the server.
  • Databases store information; SQL (MySQL) is structured, MongoDB is NoSQL.
  • Security basics include HTTPS, input validation, and authentication.
